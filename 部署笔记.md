查看 python 占用端口：
```
sudo ss -tlpn | grep python
```
参数含义：
-t   TCP
-l   正在监听
-p   显示进程
-n   不解析端口名（更快）

```
socat TCP-LISTEN:9091,fork,bind=0.0.0.0 TCP:127.0.0.1:9090
```
